FERMTOOLS - 

The program for easy video card output parameters, monitoring and alerting.
Capabilities:
- Convenient display parameters for a large number of cards,
- The ability to determine which slot this card set,
- All parameters can be read from the named pipe "pipefermtools",
- Reliable monitoring algorithm parameters,
- Support for some hardware watchdog timers to automatically restart the computer when the computer freezes or fails, a video card,
- Manual reset of the context menu,
- Record fail or error events in the windows eventlog,
- Sending e-mail alerts when you restart the computer because of the video card failures.

To control the watchdog timer requires the administrator mode for downloading drivers.

Drivers are taken "as is" from the project Open Hardware Monitor. Also, this project used some parts of the code to read the parameters of graphics cards and determine the presence watchdog:
http://openhardwaremonitor.org
https://github.com/openhardwaremonitor

Suggestions and comments are welcome.

Программа для удобного вывода параметров видеокарт, мониторинга и оповещения.
Возможности:
- удобное отображение параметров для большого количества видеокарт,
- возможность определить в каком слоте установлена данная видеокарта,
- Все параметры можно читать из поименованного канала (pipe) "pipefermtools",
- надежный алгоритм мониторинга параметров,
- поддержка некоторых аппаратных сторожевых таймеров для автоматического рестарта компьютера при зависании компьютера или сбое видео карты,
- ручной ресет из контекстного меню,
- запись событий о сбоях и ошибкаах в лог windows,
- отправка оповещений по электронной почте при перезагрузке компьтера из за  сбоев видеокарт.

Для управления сторожевым таймером требуется режим администратора для загрузки драйверов.

Драйверы взяты "как есть" из проекта опенхардмонитор. Также из этого проекта использованы некоторые участки кода для чтения параметров видеокарт и определения наличия сторожевых таймеров.
http://openhardwaremonitor.org
https://github.com/openhardwaremonitor
